# 00-Inbox 入口

`00-Inbox/` 是临时收集区，用来先接住还没分类的材料。

这里的内容不要求一开始就整理好，但需要定期清理：有价值的内容移动到研究、项目或文献目录；没价值的内容删除或归档。

## 整理规则

- 文献精读、论文原子笔记：移动到 `30-Literature/`。
- 正式研究主线：移动到 `10-Research/`。
- 有代码、实验、加工、测试流程的内容：移动到 `20-Projects/`。
- 原始 PDF、图片、大文件：移动到 `90-Local_Not_Upload/`。
- 过期网页剪藏：删除或移动到 `80-Archive/`。

## 每周清理清单

1. 看一遍 `Clippings/` 是否还有未处理资料。
2. 把可复用的观点改写成正式笔记。
3. 把实验、代码、样品相关内容移到项目目录。
4. 删除重复、过期或已经转写完成的临时材料。
