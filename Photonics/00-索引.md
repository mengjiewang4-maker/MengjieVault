# 光学实验学习索引

## 学习目标

本课题以复现 Gao et al. 的 Dirac-vortex topological cavities 光学实验为目标，当前核心材料包括本地实验计划、论文主文 PDF 和补充材料 PDF。学习目标不是泛泛学习光学实验，而是逐步获得复现论文 Fig. 5 及补充材料 S10-S12 的能力。

第一阶段复现目标：

1. 理解并复现 silica-cladded SOI Dirac-vortex cavities 的被动光学测试逻辑。
2. 对应论文 Fig. 5，测得不同 winding number、不同 vortex size 下的光谱、远场、$Q$ 和 FSR。
3. 对应 SI Fig. S11，理解并搭建或等效实现 cross-polarization reflection setup。
4. 对应 SI Fig. S12，理解 $m_0$ 和 $R$ 改变时 $Q$ 与共振波长如何变化。

因此，本仓库的笔记必须始终服务于一个问题：怎样把论文中的实验图变成自己能执行、能排错、能解释的数据流程。

可验证的能力标准：

1. 能把论文 Fig. 5 拆成可执行的实验任务：样品参数、光路、光谱、远场、统计图。
2. 能解释每个光学测试步骤解决的问题，而不只是复述流程。
3. 能从共振谱中提取 $\lambda_0$、$\Delta\lambda$、$Q$ 和 FSR，并判断拟合是否可靠。
4. 能用近场、远场和背景对照区分真实 Dirac-vortex 腔模、普通散射和仪器伪影。
5. 能根据异常现象提出排查顺序，例如找不到峰、峰漂移、远场图样不清晰。
6. 能把样品参数、光路参数、原始数据和结论组织成可复现的实验记录。

为什么需要论文和 SI：

1. 主文 Fig. 5 给出复现实验的核心目标：silica-cladded SOI 样品、不同 winding number 的光谱和远场、不同 vortex size 下的 $Q$ 与 FSR。
2. 主文说明实验样品使用 220 nm silicon layer、underneath SiO2 cladding，典型参数包括 $\alpha=4$、$m_0=50\,\mathrm{nm}$、$a=490\,\mathrm{nm}$。
3. SI Fig. S11 给出 cross-polarization reflection setup，是理解“如何在同一波长下分离泵浦和样品信号”的关键。
4. SI Fig. S12 给出 $m_0$ 和 $R$ 的实验扫描，是后续做参数统计和判断趋势的依据。

读完这些资料后应获得：

1. 明确自己要复现哪几张图。
2. 明确每张图需要哪些原始数据。
3. 明确实验失败时优先排查光路、样品、偏振还是数据处理。
