# Vortex nanolaser 与 Dirac-vortex cavity 对比

两篇论文都使用拓扑缺陷构造光学腔，但目标不同。

| 维度 | Dirac-vortex cavity | Photonic disclination nanolaser |
|---|---|---|
| 缺陷 | Dirac mass vortex | disclination defect |
| 理论 | Jackiw-Rossi zero mode | HOTI/disclination state |
| 模式体积 | 可大面积扩展 | 波长量级小体积 |
| 核心优势 | 大 FSR、单模 PCSEL 前景 | 直接发射 OAM/vortex beam |
| 实验目标 | 被动腔光学性质 | 主动纳米激光 |

一句话：

> Dirac-vortex cavity 解决“大面积单模”，photonic disclination nanolaser 解决“小体积涡旋发射”。
