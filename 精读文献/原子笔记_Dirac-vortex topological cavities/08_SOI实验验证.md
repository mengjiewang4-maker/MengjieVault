# SOI 实验验证了什么

实验平台：

- SOI（silicon-on-insulator，绝缘体上硅）。
- 220 nm silicon layer（硅层）。
- 工作在 telecommunication wavelength（通信波长）附近。
- 使用 electron-beam lithography（电子束光刻）和 dry etching（干法刻蚀）。

关键验证：

1. $w=+1,+2,+3$ 时，中隙拓扑模数量分别对应 1、2、3。
2. 远场图案与模拟相符，说明模式识别可靠。
3. Q 随模式面积增大而上升，最后受加工误差限制在 $10^4$ 到 $10^5$。
4. FSR 随模式体积接近 $V^{-1/2}$ 缩放。
5. 50 μm Dirac-vortex cavity 的 FSR 为 8.22 nm，明显大于相同模式体积 Fabry-Perot 腔的 1.28 nm。

没有验证的内容：

> 本文还没有实现真正的拓扑 PCSEL 激光器，只是验证了 Dirac-vortex cavity 的关键光学性质。
