# 无限深势阱

## 物理意义
无限深势阱描述粒子被限制在区间 `0 < x < L` 内运动，阱外势能视为无穷大，因此粒子不能出现在边界外。波函数在边界必须为零：

\[
\psi(0)=0,\quad \psi(L)=0.
\]

这意味着只有满足边界条件的驻波才能存在，因此允许能量不是连续的，而是离散的本征值。

## 数学推导
在阱内，定态方程为

\[
-\psi''(x)=E\psi(x).
\]

令

\[
E = k^2,
\]

则方程化为

\[
\psi''(x)+k^2\psi(x)=0.
\]

通解是

\[
\psi(x)=A\sin(kx)+B\cos(kx).
\]

由边界条件 `\psi(0)=0` 得 `B=0`。再由 `\psi(L)=0` 得

\[
A\sin(kL)=0.
\]

要有非平凡解，必须满足

\[
kL=n\pi,\quad n=1,2,3,\dots
\]

所以

\[
k_n=\frac{n\pi}{L},\qquad E_n=\frac{n^2\pi^2}{L^2}.
\]

归一化后本征函数为

\[
\psi_n(x)=\sqrt{\frac{2}{L}}\sin\left(\frac{n\pi x}{L}\right).
\]

对应概率密度为

\[
|\psi_n(x)|^2 = \frac{2}{L}\sin^2\left(\frac{n\pi x}{L}\right).
\]

## 概率密度图像
概率密度 `|\psi_n|^2` 总是非负，并表示粒子在位置 `x` 附近被测到的相对概率密度。

- 基态 `n=1` 只有一个峰，粒子最可能出现在阱中央附近。
- 激发态 `n>1` 会出现更多节点，节点位置概率密度为零。
- 波函数正负号会变化，但概率密度不受正负号影响。

## 有限差分数值求解
把区间 `0 < x < L` 划分成网格点 `x_i`，二阶导数可以用中心差分近似：

\[
\psi''(x_i)\approx \frac{\psi_{i+1}-2\psi_i+\psi_{i-1}}{\Delta x^2}.
\]

于是方程

\[
-\psi''=E\psi
\]

可写成矩阵本征值问题

\[
H\vec{\psi}=E\vec{\psi},
\]

其中三对角矩阵 `H` 为

\[
H=\frac{1}{\Delta x^2}
\begin{pmatrix}
2 & -1 & 0 & \cdots & 0 \\
-1 & 2 & -1 & \cdots & 0 \\
0 & -1 & 2 & \cdots & 0 \\
\vdots & \vdots & \vdots & \ddots & -1 \\
0 & 0 & 0 & -1 & 2
\end{pmatrix}.
\]

对这个矩阵做数值对角化，就能得到近似本征值和本征函数，并与解析解比较。

## 代码示例
下面脚本会：

- 画出前几个态的概率密度 `|\psi_n|^2`
- 用有限差分法数值求出本征值和本征函数
- 将数值结果和解析结果画在同一张图里
- 打印本征值误差

```python
import numpy as np
import matplotlib.pyplot as plt

L = 1.0
num_points = 200
num_states = 3

def analytic_energy(n, length):
    return (n**2 * np.pi**2) / (length**2)

def analytic_wavefunction(n, x, length):
    return np.sqrt(2.0 / length) * np.sin(n * np.pi * x / length)

dx = L / (num_points + 1)
x_num = np.linspace(dx, L - dx, num_points)

diagonal = np.full(num_points, 2.0 / dx**2)
off_diagonal = np.full(num_points - 1, -1.0 / dx**2)
H = np.diag(diagonal) + np.diag(off_diagonal, 1) + np.diag(off_diagonal, -1)

e_vals, e_vecs = np.linalg.eigh(H)
e_vecs /= np.sqrt(np.sum(np.abs(e_vecs) ** 2, axis=0) * dx)

x = np.linspace(0.0, L, 1000)
fig, axes = plt.subplots(2, 1, figsize=(8, 8))

for n in range(1, num_states + 1):
    psi = analytic_wavefunction(n, x, L)
    axes[0].plot(x, np.abs(psi) ** 2, label=f"|psi_{n}|^2")

axes[0].set_title("Probability densities")
axes[0].legend()
axes[0].grid(True, alpha=0.3)

for n in range(1, num_states + 1):
    psi_a = analytic_wavefunction(n, x_num, L)
    psi_n = e_vecs[:, n - 1]
    if psi_n[0] < 0:
        psi_n = -psi_n
    axes[1].plot(x_num, psi_a, label=f"analytic psi_{n}")
    axes[1].plot(x_num, psi_n, "--", label=f"numeric psi_{n}")

axes[1].set_title("Finite-difference comparison")
axes[1].legend(ncol=2)
axes[1].grid(True, alpha=0.3)

plt.tight_layout()
plt.show()

for n in range(1, num_states + 1):
    print(n, analytic_energy(n, L), e_vals[n - 1])
```
