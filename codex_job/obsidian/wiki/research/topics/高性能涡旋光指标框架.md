# 高性能涡旋光指标框架

这页不是比较某一篇论文，而是给你以后所有相关页面提供统一评价语言。

## 为什么必须先建这页

如果没有统一指标，你后面会不断在这些词之间来回切换：

- 高 `Q`
- 低阈值
- 大功率
- 大面积
- 单模
- 小模体积
- 模式纯度

它们彼此并不等价，很多时候还是互相拉扯的。

## 核心评价维度

| 维度 | 它回答什么问题 | 对你的最终目标意味着什么 |
| --- | --- | --- |
| `Q` | 腔模损耗低不低 | 决定模式质量、阈值潜力、场增强能力 |
| mode volume | 模式压得紧不紧 | 影响局域增强、Purcell 效应与腔模紧凑性 |
| 阈值 | 多容易起振 / 激射 | 反映器件效率和腔模质量的一部分 |
| 输出功率 / 功率密度 | 真正能输出多少 | 直接对应“大功率”目标 |
| 出光面积 / 模式扩展性 | 模式能否放大而不坏掉 | 直接对应 Dirac-vortex 路线的潜力 |
| 单模性 / 模竞争 | 会不会多模乱掉 | 对高质量稳定出光非常关键 |
| 拓扑荷 / 模式纯度 | 输出是不是干净的 vortex mode | 决定器件是否真的是高质量涡旋光源 |
| 偏振纹理 / 矢量特征 | 模式内部结构是否清楚可控 | 关系到验证和应用可用性 |
| 可制造性 | 结构是否容易加工复现 | 决定是否能从概念走向版图和器件 |

## 当前最重要的张力

### 1. 高 `Q` vs 大面积 / 高功率

- 强局域腔往往更容易做高 `Q`
- 但大面积扩展后，单模和模式纯度更难维持
- 这正是你需要同时看 `Dirac-vortex` 和 `disclination` 的原因

### 2. 小 mode volume vs 功率扩展

- 小 mode volume 更利于局域增强和低阈值
- 但未必天然适合大面积高功率输出
- `disclination` 路线在这里通常更强，但也更容易受限于尺寸和出光规模

### 3. 单模性 vs 模式扩展

- 面积越大，越要警惕模竞争和简并问题
- `Dirac-vortex` 路线的价值之一，就是它可能为大面积仍保持受控模式提供语言

## 用这页怎么读现有路线

### `Dirac-vortex`

- 更该优先观察：
  - 出光面积
  - 模式扩展性
  - 单模性
  - 表面发射可能性

### `disclination`

- 更该优先观察：
  - `Q`
  - mode volume
  - 阈值
  - 核心 vortex 模式是否干净

### 微环 / BIC / 其他器件

- 更适合充当参考样例：
  - 微环：可调谐 OAM 发射
  - BIC 超薄膜：超快、功率利用率、模块化部署
