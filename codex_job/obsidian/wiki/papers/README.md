# Papers

This section manages paper-processing workflow.
这个分区用于管理论文处理工作流。

- `inbox/`: papers or reading tasks not yet processed
  `inbox/`：尚未处理的论文或阅读任务
- `notes/`: structured paper notes
  `notes/`：结构化论文笔记
- `reviews/`: literature reviews and grouped reading outcomes
  `reviews/`：文献综述和分组阅读产出
- `reproduction/`: reproduction plans and replication notes
  `reproduction/`：复现计划和复现实验记录
