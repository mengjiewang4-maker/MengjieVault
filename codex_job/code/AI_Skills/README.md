# AI Skills Index

这个目录收集了从 `kepano/obsidian-skills` 安装的 Obsidian 相关 skills。

## 已安装 Skills

### `defuddle`

- 用途：把网页正文提取成干净的 Markdown，减少导航、广告和杂项内容。
- 适合：阅读文档、博客、文章、帮助页面。
- 入口：[defuddle/SKILL.md](./defuddle/SKILL.md)

### `json-canvas`

- 用途：创建和编辑 Obsidian 的 `.canvas` 文件。
- 适合：脑图、流程图、卡片连接图、可视化整理。
- 入口：[json-canvas/SKILL.md](./json-canvas/SKILL.md)

### `obsidian-bases`

- 用途：创建和编辑 Obsidian 的 `.base` 文件，配置视图、筛选、公式和汇总。
- 适合：表格视图、卡片视图、基于属性的数据组织。
- 入口：[obsidian-bases/SKILL.md](./obsidian-bases/SKILL.md)

### `obsidian-cli`

- 用途：通过 `obsidian` CLI 与正在运行的 Obsidian 实例交互。
- 适合：读写笔记、搜索、批量处理、插件调试。
- 入口：[obsidian-cli/SKILL.md](./obsidian-cli/SKILL.md)

### `obsidian-markdown`

- 用途：编写和修改符合 Obsidian 语法的 Markdown。
- 适合：wikilinks、embeds、callouts、frontmatter、标签与注释。
- 入口：[obsidian-markdown/SKILL.md](./obsidian-markdown/SKILL.md)

### `zotero-obsidian-topology-notes`

- 用途：把论文片段和 `@citekey` 转成面向拓扑光子学/物理主题的 Obsidian 原子笔记，并打通 Zotero 跳转与知识库双向链接。
- 适合：文献摘录原子化、Citation Key 锚定、公式抽取、Python/NumPy 映射、保存到 `概念/` 与回链 `来源/`。
- 入口：[zotero-obsidian-topology-notes/SKILL.md](./zotero-obsidian-topology-notes/SKILL.md)

## 推荐用法

- 处理网页内容时，优先看 `defuddle`
- 编辑普通 Obsidian 笔记时，优先看 `obsidian-markdown`
- 需要命令行操作 Obsidian 时，优先看 `obsidian-cli`
- 处理 `.canvas` 文件时，使用 `json-canvas`
- 处理 `.base` 文件时，使用 `obsidian-bases`
- 处理论文片段到 Zotero/Obsidian 原子卡的整理时，优先看 `zotero-obsidian-topology-notes`
