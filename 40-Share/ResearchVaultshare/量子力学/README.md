# 量子力学仓库说明

本仓库用于长期整理量子力学学习材料，入口是 [[00-索引]]。

## 当前结构

- `00-索引.md`：总目录、学习路径、当前进度。
- `01-课程主线/`：正式课程笔记，按编号推进。
- `10-概念卡片/`：一个文件只讲一个概念，方便双链复用。
- `20-公式推导/`：公式来源、推导步骤和适用条件。
- `30-练习与反馈/`：练习题、作答、批改和错题复盘。
- `40-模拟与代码/`：Python 可视化、数值模拟和交互演示。
- `90-资料与剪藏/`：论文、网页、教材、维基百科等原始资料。
- `99-协作与维护/`：协作规范和仓库维护说明。
- `figures/`、`assets/`、`attachments/`：图片、素材和附件。

## 命名原则

课程主线使用 `01-xxx.md`、`02-xxx.md` 这类编号名称。概念卡片使用清晰概念名，例如 `波函数.md`、`能级.md`。

具体协作流程见 [[99-协作与维护/00-协作规范]]。
