## 使用方式

1. 在 [[演示/量子力学/01|第 01 课]] 末尾填写「你的作答区」。
2. 让 Codex 批改，批改结果会写入「你的反馈区」。
3. 达标后说“继续下一课”，系统按 [[演示/量子力学/00-索引|学习索引]] 生成下一课。
4. 新资料先放入 `raw/` 或 `Clippings/`，再生成 `wiki/sources/` 来源摘要。
