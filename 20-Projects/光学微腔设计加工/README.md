# 光学微腔设计加工

这个目录保存光学微腔设计、版图生成、加工文件、实验报告和测试数据。它属于 `20-Projects/`，因为这里不仅是笔记，还包含代码、脚本、设备文件和实验产物。

## 当前结构

| 目录 | 内容 |
|---|---|
| `10-设计建模代码/` | 设计、建模、拓扑缺陷模式分析和旋错腔相关代码。 |
| `20-GDS版图与生成脚本/` | GDS 版图生成脚本、历史版图目录和版图相关实验脚本。GDS 是芯片/微纳加工常用的版图文件格式。 |
| `30-加工文件/` | 电子束曝光或加工相关文件，例如 `.jdf`、`.sdf`、`.mgn`。 |
| `40-实验报告与汇报/` | 实验计划、实验记录、实验手册、阶段汇报、截图和 PPT。 |
| `90-本地实验数据_不上传/` | 原始实验数据、SEM 图、光学测试数据等大文件，只保留在本机。 |

## 内容关系

| 目录或文件 | 内容 |
|---|---|
| `10-设计建模代码/旋错腔/` | 旋错腔、Dirac-vortex、拓扑缺陷模式相关的建模、版图生成和旧版脚本。 |
| `20-GDS版图与生成脚本/gds2025/` | 2025 年的版图生成脚本和相关历史记录。 |
| `20-GDS版图与生成脚本/gds/` | 2025-2026 年延续的 GDS 生成脚本和版图输出目录。 |
| `30-加工文件/6300编译文件/` | 6300 设备相关编译文件，例如 `.jdf`、`.sdf`、`.mgn`。 |
| `40-实验报告与汇报/实验报告/` | 实验计划、实验记录、实验手册和阶段性报告。 |

## 上传规则

- 上传：核心代码、文本脚本、小型加工配置文件、项目说明。
- 不上传：`90-本地实验数据_不上传/`、GDS 二进制文件、Word/PPT、截图、原始文献 PDF、本地 Python 环境。
- 大文件仍保留在本地目录中，只是被 Git 忽略；以后需要备份时建议放到外部硬盘、网盘或专门的数据仓库。
