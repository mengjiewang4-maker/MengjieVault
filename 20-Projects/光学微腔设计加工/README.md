# 光学微腔设计加工

这个目录保存光学微腔设计、版图生成、加工文件、实验报告和测试数据。它属于 `20-Projects/`，因为这里不仅是笔记，还包含代码、脚本、设备文件和实验产物。

## 内容关系

| 目录或文件 | 内容 |
|---|---|
| `旋错腔/` | 旋错腔、Dirac-vortex、拓扑缺陷模式相关的建模、版图生成和旧版脚本。 |
| `6300编译文件/` | 电子束曝光或加工相关的编译文件，例如 `.jdf`、`.sdf`、`.mgn`。 |
| `gds文件/` | 历史 GDS 版图、生成脚本、仿真或版图输出。GDS 是版图文件格式，通常体积较大。 |
| `实验报告/` | 实验计划、实验记录、实验手册和阶段性报告。 |
| `DATA/` | 原始实验数据、SEM 图、光学测试数据等。 |
| `拓扑光子晶体微腔的 SOI 加工优化及旋错态复现计划.pptx` | 项目计划汇报。 |

## 上传规则

- 上传：核心代码、文本脚本、小型加工配置文件、项目说明。
- 不上传：`DATA/`、GDS 二进制文件、Word/PPT、截图、原始文献 PDF、本地 Python 环境。
- 大文件仍保留在本地目录中，只是被 Git 忽略；以后需要备份时建议放到外部硬盘、网盘或专门的数据仓库。
