# 版本差异分析

## 对比对象

| 对象 | 当前位置 | 原路径 | 类型 | 是否与 disclination 主线相关 |
|---|---|---|---|---|
| 当前主脚本 | `current/disclination_vortex_gds_current.py` | `disclination_vortex_gds.py` | Python | 是 |
| 历史探索 notebook | `archive/old_versions/disclination_vortex_lattice_exploration_20260516.ipynb` | `disclination_vortex_1.ipynb` | Notebook | 是，偏晶格推导和可视化 |
| 历史模板脚本 | `archive/old_versions/generate_dirac_vortex_wm1_R100_template_v01.py` | `gds_template_1.py` | Python | 否，属于 Dirac/vortex 三角孔模板 |

## 关键差异

| 对比项 | 当前主脚本 | 历史探索 notebook | 历史模板脚本 |
|---|---|---|---|
| 结构参数 | `a=0.554/0.559 um`；`r/a=0.20/0.15`；`R=60` 和 `R_half=30` | `R=10`；蜂窝/六边形晶格点推导；主要用于画散点 | `a=0.490`；`m0=0.15`；`R=125`；`core=60`；`s=0.32`；`w=-1` |
| 输出文件名 | `mjYYYYMMDD_01.gds` 到 `mjYYYYMMDD_06.gds` | 不输出 GDS | `wmjYYYYMMDDw-1R100.gds` |
| GDS cell 命名 | `DISCLINATION_VORTEX` | 无 GDS cell | `SAMPLE` |
| 是否加入 mark | 是，写入 `a..._r...` 参数标识 | 否 | 是，写入 `w=-1` |
| 是否加入阵列 | 是，5 个扇区复制形成 disclination 空气孔阵列 | 是，生成并绘制晶格点 | 是，生成 Dirac/vortex 三角孔阵列 |
| 是否加入 dose matrix | 否 | 否 | 否 |
| 是否加入 PEC 标记 | 否 | 否 | 否 |
| disclination 参数 | 4 个原始扇区 + 1 个添加扇区；最终 5 扇区；角度比例 0.8；半径比例 0.8 | 有角度映射探索，但不是最终 GDS 脚本 | 无 disclination 参数 |
| 版图尺寸 | R=60 全尺寸和 R=30 半尺寸 | R=10 可视化 | R=125；绘制半径约 0.8R |
| 明显 bug | 未发现明显语法问题；风险是移动后输出路径随 `__file__` 变化 | 有一次列表二维切片报错记录，说明 notebook 运行状态不稳定 | 代码可读但注释中 `a = 0.490 # lattice constant in nm` 与 GDS 微米量级可能不一致，需要复核单位 |
| 重复代码 | 与历史模板重复了矢量文字 `draw_vector_text` 逻辑 | 与主脚本有晶格点生成/旋错思想重叠 | 与主脚本重复了 mark 绘制逻辑，但结构完全不同 |

## 结论

- 当前主线应以 `current/disclination_vortex_gds_current.py` 为基准。
- `disclination_vortex_lattice_exploration_20260516.ipynb` 适合作为理解 disclination 映射的历史推导，不适合作为 EBL 版图生成入口。
- `generate_dirac_vortex_wm1_R100_template_v01.py` 是另一类 Dirac/vortex 模板，不建议混入 batch13 disclination 版图流程。
