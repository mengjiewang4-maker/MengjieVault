# 推荐主版本说明

## 推荐主文件

`current/disclination_vortex_gds_current.py`

原始来源：`disclination_vortex_gds.py`

## 推荐理由

1. 能生成最新 6 个 GDS：`mj20260517_01.gds` 到 `mj20260517_06.gds`。
2. 参数清晰：`PARAM_SETS` 明确列出 3 组周期和孔半径比例，每组自动生成全尺寸和半尺寸两个 GDS。
3. 输出文件名明确：`mjYYYYMMDD_01.gds` 到 `mjYYYYMMDD_06.gds`。
4. disclination 参数集中：4 个原始扇区 + 1 个添加扇区，最终 5 扇区，角度比例和半径比例都是 0.8。
5. 已自动生成参数记录 Markdown，便于和样品数据库对应。
6. 最适合后续加入 batch13 的 mark、dose matrix 和 PEC 对应表。

## 目前风险

- 脚本使用 `Path(__file__).resolve().parent` 作为输出根目录。整理后脚本位于 `current/`，如果直接运行，默认会输出到 `current/YYYYMMDD`，而不是原来的 `vippy/YYYYMMDD`。
- 当前脚本没有 dose matrix，也没有 PEC 区域标记。
- 当前脚本没有显式版本号，只有日期戳和文件序号。
- 当前脚本没有防止重复运行造成实验对应关系混淆的机制。

## 后续应如何修改

- 不直接改 `current/disclination_vortex_gds_current.py`。
- 先复制为 `disclination_vortex_gds_YYYYMMDD_v01.py` 或 `generate_batch13_disclination_dose_matrix_v01.py`。
- 在新版本中加入清晰的 batch 编号、区域编号、dose matrix 和 PEC 状态标记。
- 固定输出目录，例如 `outputs/gds/batch13_YYYYMMDD/`，避免脚本移动后输出路径变化。
- 每次生成 GDS 后更新 `Py_GDS_对应关系.md` 和样品数据库中的 GDS 参数表。

## 不建议直接使用的历史版本

- `archive/old_versions/generate_dirac_vortex_wm1_R100_template_v01.py`：这是 Dirac/vortex 三角孔模板，不是当前 disclination cavity 主线。
- `archive/old_versions/disclination_vortex_lattice_exploration_20260516.ipynb`：这是推导和可视化 notebook，没有 GDS 写出流程，且有历史运行报错记录。
