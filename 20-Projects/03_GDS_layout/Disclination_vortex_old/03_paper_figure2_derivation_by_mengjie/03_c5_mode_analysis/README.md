# C5 Mode Analysis

本目录保存 C5/C6 disclination 模式分析脚本。

重点内容包括：

- C5 cell-center 和 bond-center 几何探索
- 零模分析
- IPR 局域性分析
- KDTree 近邻搜索
- 中心和边缘耦合参数调节

`IPR` 是 inverse participation ratio，常用于判断模式是否局域在少数位置。
