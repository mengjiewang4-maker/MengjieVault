# Core Square Lattice Pipeline

本目录保存方形晶格紧束缚模型的核心流程。

建议阅读顺序：

1. `square_lattice_disclination_geometry.py`
2. `tight_binding_hamiltonian_builder.py`
3. `tight_binding_eigensolver.py`
4. `mode_angular_momentum_analysis.py`
5. `lattice_mode_plotting_helpers.py`
6. `square_lattice_tb_demo_pipeline.py`

`tight-binding` 是紧束缚模型，用矩阵描述不同格点之间的耦合。
