# Disclination vortex 晶格分类笔记

> [!note] 当前依据
> 本笔记按当前目录中实际存在的文件分类。因为部分文件已经删改或移动，旧入口不再使用；例如 `comsol/export_comsol.py`、`comsol/export_lumerical.py`、`comsol/resonators.csv` 当前在 `comsol/` 目录下，六角晶格代码也有一份在 `六边形/` 目录下。

## 分类原则

- **六角晶格 / 六边形裁剪**：代码里通常会出现 `is_point_in_hexagon()`、`hexagon radius`、六边形边界判断，或者使用蜂窝/三角晶格基矢。
- **四角晶格 / 方形晶胞 / 四原子 BBH 类**：代码里通常会出现 `generate_structure()`、`generate_lattice()`、`q1_atoms`、`basis = [[cx-delta, cy-delta], ...]`。这表示一个方形晶胞里放 4 个点，再做角度映射和旋转拼接。
- **混合类**：边界是四边形或四角裁剪，但内部点位仍然使用蜂窝/三角晶格基矢。

## 混合类：四边形边界 + 蜂窝/三角基矢

| 文件 | 说明 |
| --- | --- |
| `disclination_vortex_gds.py` | 当前文件里使用 `is_point_in_quadrilateral()`，所以边界是四边形/四角裁剪；但内部仍使用 `a1 = [sqrt(3)/2, 1/2]`、`a2 = [sqrt(3)/2, -1/2]` 这样的蜂窝/三角晶格基矢。 |

这类不要简单归到“标准四原子 BBH 方形晶胞”。更准确的说法是：**四边形边界裁剪的蜂窝/三角基矢结构**。

## 简短结论

- 要找**六角晶格**，优先看 `六边形/` 目录、`disclination_vortex_1.ipynb`，以及 `20260420/` 这批 GDS。
- 要找**四角晶格/四原子 BBH 模型**，优先看 `figure2/` 目录、`todo/square_lattice_volterra_sector_geometry.py`、`todo/square_lattice_structure_export_pipeline.py`、`todo/pythtb_2d_ssh_bbh_model_builder.py`。
- 当前根目录的 `disclination_vortex_gds.py` 是混合情况：四边形边界，但内部仍是蜂窝/三角基矢。
