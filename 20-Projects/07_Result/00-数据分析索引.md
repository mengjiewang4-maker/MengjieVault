# 数据分析索引

## 文件夹用途

这里用于记录从原始测试数据到结果图的处理过程。重点不是只保存最终图片，而是保存“数据从哪里来、怎么处理、得到什么结论”。

## 推荐记录什么内容

- 光谱分析：峰位、线宽、Q 因子、拟合方法。Q 因子表示共振峰的品质因数，常用于衡量微腔损耗。
- 远场分析：远场图、角分布、模式形貌。
- 偏振分析：偏振依赖、偏振角扫描。
- 干涉图分析：相位、拓扑荷、涡旋特征。
- Python 脚本记录：脚本用途、输入输出、版本变化。

## 和其他文件夹如何双链关联

- 分析记录链接到原始测试记录。
- 分析结果链接回器件页和加工批次。
- 可用图表链接到论文与汇报文件夹。
- 分析异常链接到问题排查记录。

## 推荐双链格式

```markdown
器件：[[DV-DISC-2026-001]]
测试记录：[[测试-2026-05-14-DV-DISC-2026-001]]
原始数据：[[数据-2026-05-14-DV-DISC-2026-001]]
分析记录：[[分析-2026-05-14-DV-DISC-2026-001]]
论文图：[[图-DV-DISC-2026-001-Q因子拟合]]
```
