# 下一版 EBL 调整建议

## 1. 当前问题

batch12_Si_EBL 的 SEM 结果显示曝光过度。

## 2. 调整方向

优先调整剂量，而不是立即修改结构参数。

建议顺序：

1. 整理 6 个 GDS 的结构参数；
2. 建立 GDS-剂量-SEM 图像对应表；
3. 测量关键尺寸；
4. 判断过曝程度；
5. 设计下一版更低剂量矩阵；
6. 再决定是否修改 GDS 尺寸补偿。

## 3. 下一版剂量矩阵候选

暂定：

| 剂量组 | 说明 |
|---|---|
| 25% | 明显低于当前最低剂量 |
| 30% | 低剂量测试 |
| 35% | 中低剂量测试 |
| 40% | 接近当前最低剂量 |
| 45% | 与当前最低剂量对照 |

注意：这只是初步建议，最终剂量需要根据 SEM 尺寸测量结果确定。
