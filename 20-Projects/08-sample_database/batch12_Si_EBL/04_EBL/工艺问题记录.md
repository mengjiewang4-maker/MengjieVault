# batch12_Si_EBL 工艺问题记录

## 1. 已发现问题

- SEM 初步观察显示整体曝光过度。

## 2. 可能相关工艺参数

| 参数 | 当前设置 | 可能影响 |
|---|---|---|
| 胶 | AR-P 6200.09 | 胶厚和灵敏度会影响最佳剂量窗口 |
| 旋涂 | 500 rpm 5 s + 2000 rpm 60 s | 影响胶厚，进而影响显影和结构成形 |
| 前烘 | 150 ℃ 60 s | 影响胶膜状态 |
| 曝光剂量 | 45%、55%、65% | 当前判断偏高 |
| 显影 | 乙酸乙酯 60 s | 影响图形打开程度和线宽 |
| 后烘 | 130 ℃ 3 min | 可能影响结构变形或稳定性 |

## 3. 待确认

- 不同 GDS 的图形密度是否差异明显；
- 过曝是否在 45%、55%、65% 三个剂量中均明显；
- 中心 disclination 区域是否比边缘区域更严重；
- 是否需要做邻近效应修正。
