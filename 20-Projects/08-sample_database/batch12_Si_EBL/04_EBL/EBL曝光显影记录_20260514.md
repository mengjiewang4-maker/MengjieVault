# batch12_Si_EBL EBL曝光显影记录

## 1. 实验目标

复现 Nature Photonics 2024 Fig.2 disclination cavity，测试 6 个 GDS 文件在不同剂量下的曝光显影效果。

## 2. 工艺流程

1. 涂胶
2. 前烘
3. EBL 曝光
4. 显影
5. IPA 定影
6. 后烘
7. SEM 观察

## 3. 工艺参数

| 项目 | 参数 |
|---|---|
| 胶 | AR-P 6200.09 |
| 旋涂 | 500 rpm 5 s + 2000 rpm 60 s |
| 前烘 | 150 ℃ 60 s |
| 曝光剂量 | 45%、55%、65% |
| 显影液 | 乙酸乙酯 |
| 显影时间 | 60 s |
| 定影 | IPA |
| 后烘 | 130 ℃ 3 min |

## 4. 观察结果

SEM 显示曝光过度。

## 5. 初步判断

当前剂量窗口偏高，下一版 EBL 需要降低剂量或重新设计更低剂量范围。
