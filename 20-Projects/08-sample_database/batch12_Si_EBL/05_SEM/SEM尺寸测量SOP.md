# batch12_Si_EBL SEM尺寸测量SOP

SOP 是 Standard Operating Procedure 的缩写，意思是“标准操作流程”。这份记录用于保证每张 SEM 图的测量方法尽量一致。

## 1. 测量前准备

1. 打开 SEM tif 原图，不修改原图；
2. 确认图中标尺是否清楚；
3. 在 [SEM文件名索引表.md](SEM文件名索引表.md) 中确认当前文件名；
4. 在 [GDS-剂量-SEM对应表.md](GDS-剂量-SEM对应表.md) 中填写对应的 GDS 编号和 Dose。

## 2. 建议测量参数

| 参数 | 测量目的 | 建议测量位置 |
|---|---|---|
| 周期 a | 判断结构周期是否接近设计值 | 中心区、边缘区 |
| 孔径或线宽 | 判断曝光剂量是否偏高或偏低 | 中心区、边缘区 |
| 中心 disclination 区域尺寸 | 判断缺陷腔是否成形 | 中心缺陷区域 |
| 边缘形貌 | 判断是否有残胶、塌陷或过曝 | 阵列边缘 |

## 3. 每张图的测量建议

- 每个 GDS、每个 Dose 至少测 3-5 个位置；
- 中心区和边缘区最好分开记录；
- 如果图中结构明显变形，先记录形貌问题，再决定是否继续量尺寸；
- 如果图中标尺不清楚，不要强行测量，先标记为“标尺待确认”。

## 4. 记录格式

把测量结果填写到 [SEM尺寸测量表.md](SEM尺寸测量表.md)。

| 字段 | 填写说明 |
|---|---|
| SEM文件名 | 使用完整 tif 文件名 |
| GDS编号 | GDS_01 到 GDS_06 |
| Dose | 45%、55%、65% |
| 位置 | 中心、边缘、过渡区等 |
| 设计周期a/nm | 来自 GDS 参数表 |
| 实测周期a/nm | 从 SEM 图测量得到 |
| 偏差/nm | 实测值 - 设计值 |
| 相对误差/% | 偏差 / 设计值 × 100% |
| 观察结论 | 例如过曝、结构粘连、孔径偏小等 |

## 5. 标注图归档

- 原图不要修改；
- 测量后导出的带标尺或标注图片放入 `05_SEM/标注图/`；
- 如果导出多个版本，文件名建议包含 SEM 原文件名、测量位置和日期。

示例：

```text
BATCH12_20260514_EBL_S1_P4_1_center_measure_20260519.png
```
