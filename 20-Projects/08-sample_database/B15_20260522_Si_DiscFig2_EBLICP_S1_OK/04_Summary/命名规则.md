# 样品与 SEM 命名规则

## 1. 样品命名规则

推荐格式：

```text
B批次_日期_基底_结构来源_工艺阶段_样品编号_结果状态
```

当前样品：

```text
B15_20260522_Si_DiscFig2_EBLICP_S1_OK
```

旧名映射：

| 旧样品名 | 新样品名 |
|---|---|
| BATCH15_20260522_EBL_ICP_RIE_S1 | B15_20260522_Si_DiscFig2_EBLICP_S1_OK |

## 2. SEM 文件命名规则

推荐格式：

```text
B15_20260522_Si_DiscFig2_a559r0p2n3p4_D90_R01_center_30k_OK_001.tif
```

字段解释：

| 字段 | 含义 |
|---|---|
| B15 | Batch15 |
| 20260522 | 日期 |
| Si | 基底 |
| DiscFig2 | Disclination cavity, Nature Photonics 2024 Fig.2 |
| a559r0p2n3p4 | GDS 关键参数 |
| D90 | EBL 剂量 90% |
| R01 | 区域编号 |
| center | SEM 拍摄位置 |
| 30k | SEM 倍率 |
| OK | 结果 |
| 001 | 第几张图 |

## 3. 区域编号规则

| 区域编号 | 含义 |
|---|---|
| R01 | 当前第一个明确 GDS 区域 |
| R02 | 后续第二个 GDS、第二个 Dose 或第二个结构区域 |
| R03 | 后续第三个 GDS、第三个 Dose 或第三个结构区域 |

当前 `R01`：

| 区域编号 | 对应 GDS | Dose | 基底 | 结果 |
|---|---|---|---|---|
| R01 | `disclination_a559_r0p2_n3p4_extended_100um.gds` | 90% | Si | OK |

## 4. 命名目标

以后看到 SEM 文件名，要能直接反查：

- 基底是 Si 还是 SOI；
- 对应哪个 GDS；
- Dose 是多少；
- 对应哪个样品；
- 属于哪个版图区域；
- 拍摄位置和倍率；
- 结果是否可用。
