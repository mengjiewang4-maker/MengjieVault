## 最小实战流程

拿到一个陌生项目后，先不要运行任何脚本，按这个顺序做：

```bash
find . -name "*.py" | sort
rg -n "if __name__|import |from .* import|write_gds|savefig|to_csv|np.savetxt|plt.show|print\\(" .
find . -maxdepth 3 -type f \\( -name "*.gds" -o -name "*.csv" -o -name "*.png" -o -name "*.md" \\) | sort
```

然后回答 5 个问题：

1. 哪些文件像入口？
2. 哪些文件只是函数库？
3. 哪些文件会写出 GDS/CSV/PNG？
4. 哪些 import 指向项目内部文件？
5. 运行前是否需要切换到特定目录？
