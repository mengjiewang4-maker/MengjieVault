# 第5章 中心力场

## 章节定位

本章把三维定态问题化为“角向角动量本征问题 + 径向一维有效势问题”。核心是：中心势 $V(r)$ 的球对称性保证轨道角动量守恒，使能量本征态可用 $n_r,l,m$ 或 $n,l,m$ 等量子数标记。

## 目录结构

- 5.1 [[中心力场]] 中粒子运动的一般性质
  - 角动量守恒与径向方程
  - 径向波函数在 $r=0$ 邻域的渐近行为
  - 两体问题化为约化质量单体问题
- 5.2 无限深球方势阱
  - $s$ 态与一般 $l$ 态
  - 球 Bessel 函数与边界条件 $j_l(ka)=0$
  - 有限球半径导致离散能级，$a\to\infty$ 过渡到连续谱
- 5.3 三维各向同性谐振子
  - 径向方程与合流超几何函数
  - 能级 $E_N=(N+3/2)\hbar\omega$
  - 球坐标基与 Cartesian 基之间的么正变换
- 5.4 [[氢原子]]
  - Coulomb 势径向方程
  - Bohr 能级、简并度与径向概率
  - 轨道磁矩与类氢离子

## 核心公式

| 主题 | 公式 | 含义 |
|---|---|---|
| 中心势 Hamilton 量 | $\hat H=\frac{\hat p^2}{2\mu}+V(r)$ | 两体问题可化为约化质量 $\mu$ 的单体问题 |
| 角动量守恒 | $[\hat{\mathbf L},\hat H]=0$ | 球对称性对应轨道角动量守恒 |
| 分离变量 | $\psi_{Elm}(r,\theta,\phi)=R_{El}(r)Y_l^m(\theta,\phi)$ | 角向由 [[球谐函数]] 给出 |
| 径向方程 | $R''+\frac{2}{r}R'+\left[\frac{2\mu}{\hbar^2}(E-V)-\frac{l(l+1)}{r^2}\right]R=0$ | 三维中心势的径向本征方程 |
| 约化径向函数 | $u(r)=rR(r)$ | 将径向方程写成一维形式 |
| 有效势 | $V_{\mathrm{eff}}(r)=V(r)+\frac{\hbar^2l(l+1)}{2\mu r^2}$ | 离心势垒抑制小 $r$ 区域 |
| 原点边界条件 | $u(0)=0$ | 物理解要求在原点正则 |
| 球方势阱 | $j_l(ka)=0,\quad E_{n_rl}=\frac{\hbar^2\alpha_{n_rl}^2}{2\mu a^2}$ | $\alpha_{n_rl}$ 为球 Bessel 函数零点 |
| 三维谐振子 | $E_N=(N+\frac32)\hbar\omega,\quad N=2n_r+l$ | 能级只依赖 $N$ |
| 三维谐振子简并度 | $f_N=\frac12(N+1)(N+2)$ | 高于一般中心势的 $2l+1$ 简并 |
| 氢原子能级 | $E_n=-\frac{\mu e^4}{2\hbar^2n^2}=-\frac{e^2}{2a_0n^2}$ | Coulomb 势束缚态能级 |
| 氢原子主量子数 | $n=n_r+l+1$ | 能级只依赖 $n$ |
| 氢原子简并度 | $f_n=n^2$ | 不计自旋时同一 $n$ 壳层的空间态数 |
| 类氢离子 | $E_n=-\frac{\mu Z^2e^4}{2\hbar^2n^2}$ | 核电荷 $Ze$ 使能级按 $Z^2$ 缩放 |

## 概念澄清

- 一般中心势中，能量通常依赖 $n_r,l$，而不依赖 $m$；这给出 $2l+1$ 重磁量子数简并。
- 三维各向同性谐振子和 Coulomb 势是特殊中心势，能量只依赖更少的量子数组合，因此出现额外简并。
- 径向方程看起来像一维问题，但定义域是 $r\ge 0$，且必须满足 $u(0)=0$。
- $l=0$ 时离心势项消失；$l>0$ 时离心势垒使波函数在原点附近按 $R(r)\sim r^l$ 消失。
- 氢原子的“轨道”不是经典轨道，笔记中应优先使用径向概率分布 $|u_{nl}(r)|^2$ 和角向概率 $|Y_l^m|^2$。
