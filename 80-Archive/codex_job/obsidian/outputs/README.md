# Outputs

This layer stores final-facing material.
这一层存放面向最终表达的内容。

- `reports/`: formal writeups and experiment reports
  `reports/`：正式写作稿和实验报告
- `slides/`: presentation drafts and slide notes
  `slides/`：演示文稿草稿和幻灯片笔记
- `summaries/`: concise external-facing summaries
  `summaries/`：简洁的对外总结

Important standalone figures should go to `results/` instead of staying only in the vault.
重要的独立图像应放入 `results/`，而不是只保留在 vault 内。
