# 工作流

## 通用流程
1. 明确问题所属领域：量子力学、光学、跨领域、项目。
2. 先补充概念说明，再整理数学推导。
3. 如有必要，提供可运行的 Python 数值验证代码。
4. 将结果归档到对应的 Obsidian 目录中。

## 量子力学工作流
1. 概念放入 `obsidian/知识库/量子力学/概念/`
2. 推导放入 `obsidian/知识库/量子力学/推导/`
3. 题目与模型放入 `obsidian/知识库/量子力学/问题/`
4. 常见错误放入 `obsidian/知识库/量子力学/易错点/`
5. 实验或数值验证放入 `obsidian/知识库/量子力学/实验/`
6. 汇总内容放入 `obsidian/知识库/量子力学/总结/`

## 光学工作流
1. 基础概念放入 `obsidian/知识库/光学/概念/`
2. 器件内容放入 `obsidian/知识库/光学/器件/`
3. 加工工艺放入 `obsidian/知识库/光学/加工/`
4. 仿真内容放入 `obsidian/知识库/光学/仿真/`
5. 文献摘录放入 `obsidian/知识库/光学/文献/`
6. 实验记录放入 `obsidian/知识库/光学/实验/`

## 代码组织
- `code/量子力学/`：量子力学相关代码
- `code/光学/`：光学相关代码
- `code/共享/`：通用工具与共享模块
