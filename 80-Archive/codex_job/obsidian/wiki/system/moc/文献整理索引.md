# 文献整理索引

## 统计区

```dataviewjs
const pages = dv.pages('"来源"');
const count = (status) => pages.where(p => p.status === status).length;

dv.table(
  ["状态", "数量"],
  [
    ["to-read", count("to-read")],
    ["reading", count("reading")],
    ["reviewed", count("reviewed")],
    ["cited-in-review", count("cited-in-review")]
  ]
);
```

## 📥 待读

```dataview
TABLE file.link AS 文献, year AS 年份, tags AS 标签
FROM "来源"
WHERE status = "to-read"
SORT year DESC
```

## 📖 正在读

```dataview
TABLE file.link AS 文献, year AS 年份, tags AS 标签
FROM "来源"
WHERE status = "reading"
SORT year DESC
```

## ✅ 已精读

```dataview
TABLE file.link AS 文献, year AS 年份, tags AS 标签
FROM "来源"
WHERE status = "reviewed"
SORT year DESC
```

## 🧩 综述引用

```dataview
TABLE file.link AS 文献, year AS 年份, tags AS 标签
FROM "来源"
WHERE status = "cited-in-review"
SORT year DESC
```

## 使用说明

- 修改文献分区：打开对应 `来源/` 页面，修改 frontmatter 里的 `status`
- 可用状态：`to-read`、`reading`、`reviewed`、`cited-in-review`
- 新文献默认进入“待读”：把 `status` 设为 `to-read`
- 开始处理一篇文献后：把 `status` 改为 `reading`
- 完成摘要级整理后：把 `status` 改为 `reviewed`
- 当该文献已经被综述页稳定引用后：把 `status` 改为 `cited-in-review`
- Dataview 面板数据来源固定为 `来源/` 目录；只要来源页 frontmatter 中有正确的 `status`、`year`、`tags`，它就会自动进入对应分区
