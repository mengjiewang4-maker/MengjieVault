# System

System pages define how the vault works.
系统页面定义这个 vault 的运行方式。

- `agents/`: agent roles and assistant-facing guidance
  `agents/`：智能体角色和面向助手的说明
- `templates/`: reusable note templates
  `templates/`：可复用笔记模板
- `workflows/`: process notes and environment workflows
  `workflows/`：流程说明和环境工作流
- `moc/`: maps of content and navigation hubs
  `moc/`：内容地图和导航枢纽
