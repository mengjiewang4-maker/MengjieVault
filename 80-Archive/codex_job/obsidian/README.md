# Obsidian Vault Structure

This vault uses a mixed `raw / wiki / outputs` structure:
这个 vault 使用混合式的 `raw / wiki / outputs` 结构：

- `raw/`: imported source material and unrefined inputs
  `raw/`：导入的原始材料和未经整理的输入
- `wiki/`: structured notes, knowledge, research, and system pages
  `wiki/`：结构化笔记、知识、研究内容和系统页面
- `outputs/`: final-facing reports, slides, and summaries
  `outputs/`：面向最终表达的报告、幻灯片和总结

请使用 [[obsidian/wiki/system/moc/Vault_Map]] 作为主导航入口。
