# Code

This directory stores runnable projects, scripts, notebooks, and local tool implementations.
本目录用于存放可运行项目、脚本、笔记本以及本地工具实现。

- Keep project-specific outputs close to the code when they are part of the build or run workflow.
  项目特定输出如果属于构建或运行流程的一部分，可以保留在代码旁边。
- Move durable figures or final result files that should be referenced long term into `results/`.
  需要长期引用的稳定图像或最终结果文件应移到 `results/`。
