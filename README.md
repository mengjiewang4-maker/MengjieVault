# MengjieVault

这个仓库是当前 Obsidian 主库，用来保存研究笔记、项目代码、文献精读、分享材料和归档资料。

## 目录分层

| 目录 | 用途 |
|---|---|
| `00-Inbox/` | 临时收集区，例如网页剪藏、随手记。 |
| `10-Research/` | 正式研究笔记。当前主线是 `Topological_Photonics/` 和 `UAV/`。 |
| `20-Projects/` | 有代码、脚本、环境文件或可执行产物的项目，例如 `Project_Replication_Vortex/`、`Skills/`、`光学微腔设计加工/`。 |
| `30-Literature/` | 文献精读、原子笔记、论文汇报稿。 |
| `40-Share/` | 准备分享、演示或发布的内容。 |
| `80-Archive/` | 旧库、迁移残留、历史项目。优先只读，不作为新内容入口。 |
| `90-Local_Not_Upload/` | 本地大文件和备份，不上传 GitHub。 |

## 使用规则

- 新资料先放 `00-Inbox/`，整理后再进入研究、文献或项目目录。
- 正式研究内容优先写入 `10-Research/`。
- 代码和可复现实验流程放 `20-Projects/`。
- 旧库资料可以引用，但不要继续在 `80-Archive/` 里扩展新主线。
- PDF、Git 备份、大型输出文件放 `90-Local_Not_Upload/`，该目录已被 Git 忽略。
- `.obsidian/workspace.json` 是 Obsidian 窗口状态，已停止跟踪，不需要提交。
