## 写作规则

- 精读笔记：保留论文结构、关键图、关键公式和作者结论。
- 原子笔记：一张卡片只写一个概念、机制、公式或实验判断。
- 汇报稿：面向组会或讨论，重点写背景、方法、结果、问题和下一步。
- 可复用结论：再链接回 `10-Research/` 或 `20-Projects/` 的相关主题。

## 推荐流程

1. PDF 原件放入 `90-Local_Not_Upload/PDF/`。
2. 在 `精读文献/` 新建精读笔记。
3. 把核心概念拆成原子笔记。
4. 把与加工、仿真、测试直接相关的结论链接到 `20-Projects/`。
5. 把长期理论理解沉淀到 `10-Research/`。
