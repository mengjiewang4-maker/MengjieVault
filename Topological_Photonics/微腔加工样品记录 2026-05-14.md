# 微腔加工样品总记录

> [!info] 记录用途
> 这份记录用于把多次加工的微腔样品和后续光学测试一一对应起来。每次加工追加一个“加工批次”，光学测试时优先查“跨批次器件总表”，避免测到共振后找不到对应的 GDS、曝光、刻蚀和 SEM 信息。

## 0. 使用规则

1. 每次新加工都先在“批次索引”新增一行，再复制“新批次复制模板”到正文末尾。
2. 每批样品使用一个批次编号，推荐格式为 `MC-YYYYMMDD-NN`，例如 `MC-20260514-01`。
3. 曝光用 GDS 推荐使用 `mjYYYYMMDD_序号.gds`，例如 `mj20260420_01.gds`；具体参数必须写入 README 和本记录的 GDS 参数表。
4. 单个器件编号推荐使用 `批次编号-Dxx`，例如 `MC-20260514-01-D03`。
5. SEM、光谱、近场、远场文件名中也要带上批次编号和器件编号，便于追溯。

推荐文件夹结构：

```text
results/microcavity/
  MC-YYYYMMDD-NN/
    gds/
    exposure/
    etch/
    sem/
    optical_spectra/
    near_field/
    far_field/
    analysis/
```

## 0.1 术语说明

- GDS：版图文件，也就是送去电子束曝光或光刻的图形文件。
- 曝光：把 GDS 图形写到胶上的步骤，常见方式是电子束曝光或光刻。
- 刻蚀：把胶上的图形转移到材料里的步骤，通常要记录气体、功率、时间和刻蚀深度。
- SEM：扫描电子显微镜图，用来检查孔径、周期、边缘粗糙度和结构是否塌陷。
- chip：一小片样品。
- die：chip 上的一个区域。
- device / cavity：具体要测的单个微腔结构。
- $a$：晶格常数，即相邻周期结构的间距。
- $\alpha$：Dirac-vortex / Kekule 调制参数。
- $m_0$：最大位移，用来描述调制强度。
- $R$：vortex size，腔的尺寸参数。
- $w$：winding number，涡旋绕数。
- $r_\mathrm{hole}$：空气孔半径。
- $r_\mathrm{ratio}$：空气孔半径与晶格常数的比例，即 $r_\mathrm{hole}/a$。
