## 固定生成参数

| 参数 | 当前值 | 说明 |
|---|---:|---|
| `sample_name` | `DISCLINATION_VORTEX` | GDS 顶层 cell 名称 |
| `n_vertex` | 256 | 圆孔用 256 边形近似 |
| `angle_ratio` | 6/5 | 旋错角度重映射比例 |
| `radius_ratio` | 6/5 | 旋错半径重映射比例 |
| `layer` | `(1, 0)` | GDS 图层 |
| `dbu` | 0.001 um | GDS 数据库单位 |

## 注意事项

- 文件名里的 `r0p111`、`r0p112`、`r0p084` 是实际孔半径，单位为 um，不是比例。
- `r_ratio` 是孔半径比例，计算式为 `r_hole = r_ratio * a`。
- `R` 是六边形晶格半径，单位是晶格格点数，不是微米。
- 当前 6 个 GDS 是圆形空气孔 disclination vortex；早期 `vortex230`、`vortex260` 不应和这批参数混用。
